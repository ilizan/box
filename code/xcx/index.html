新建小程序项目

使用微信开发者工具， 输入appid 选择云开发

删除无用文件

删除miniprogram下components文件夹

删除style文件夹

再把pages文件夹内 除了index文件夹，其他文件全删掉

cloudfunctions,images文件夹里所有文件，全部删掉。

将pages/index文件夹内的user-unlogin.png图片删掉。

把index.wxml，index.wxss文件内代码删掉

把index.js文件内代码删掉，初始化index.js文件内代码。

输入page选第4或第5个

把app.wxss文件代码删掉

把app.json文件内只留index

安装vant组件库

在微信开发者工具中 miniprogram 右键 ->从终端中打开

创建package.json： 输入npm init -y -y的意思是 一路yes回车

再输入官网的npm i @vant/weapp -S --production命令

完成后 目录下会多node_modules文件夹

修改app.json

将 app.json 中的 "style": "v2" 去除

修改project.config.json

（如果构建不报错，也可以不操作本步骤）

{
  ...
  "setting": {
    ...
    "packNpmManually": true,
    "packNpmRelationList": [
      {
        "packageJsonPath": "./package.json",
        "miniprogramNpmDistDir": "./miniprogram/"
      }
    ]
  }
}

设置npm

在右上角-详情-本地设置里面，勾选使用npm模块

左上角-工具-构建npm，目录下会多miniprogram_npm文件夹

TIP

如果构建报错，可以尝试将增强编译选项关掉再尝试。

或者把 是否检查安全域名和 TLS 版本 重置下 再尝试。

使用组件

在app.json或index.json中引入组件，建议在app.json文件中加入。

"usingComponents": {
  "van-button": "@vant/weapp/button/index"
}

自定义vant的tabbar

微信官方文档

vant文档

首先创建custom-tab-bar文件夹，包含index.js index.json index.wxml

index.js

Component({
    data: {
      active: 0,
      list: [
        {
          "url": "/pages/index/index",
          "icon": "wap-home-o",
          "text": "首页"
        },
        {
          "url": "/pages/login/login",
          "icon": "user-circle-o",
          "text": "登录"
        }
      ]
    },
    methods: {
     onChange(e) {
        console.log(e,'e')
        this.setData({ active: e.detail });
        wx.switchTab({
          url: this.data.list[e.detail].url
        });
     },
     init() {
         const page = getCurrentPages().pop();
         this.setData({
        　  active: this.data.list.findIndex(item => item.url === `/${page.route}`)
         });
        }
     }
});

index.json文件

{
    "component": true,
    "usingComponents": {
      "van-tabbar": "@vant/weapp/tabbar/index",
      "van-tabbar-item": "@vant/weapp/tabbar-item/index",
      "van-icon": "@vant/weapp/icon/index"
    }
}

index.html文件

<van-tabbar active="{{ active }}" bind:change="onChange">
  <van-tabbar-item wx:for="{{ list }}" wx:key="index" icon="{{ item.icon }}">
    {{item.text}}
  </van-tabbar-item>
</van-tabbar>

app.json中增加

"tabBar": {
  "custom": true,
  "color": "#000000",
  "selectedColor": "#000000",
  "backgroundColor": "#000000",
  "list": [
    {
      "pagePath": "pages/index/index",
      "text": "首页"
    },
    {
      "pagePath": "pages/login/login",
      "text": "登录"
    }
  ]
},

最后在每个使用tabbar的页面加上这一句，不加高亮显示慢一步

onShow: function () {
    this.getTabBar().init();
},
