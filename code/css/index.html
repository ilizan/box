css 方式引入字体

方法一引入谷歌在线字体（常用）

大概谷歌谷歌 Fonts 地址：链接

  1. 在 Head 部分加入 Google Web Fonts API 中的字体链接(Font+Name 指你要使用的字体名字)：
<link
  rel="stylesheet"
  type="text/css"
  href="http://fonts.googleapis.com/css?family=Long+Cang"
/>
<!-- <link rel="stylesheet" type="text/css" href="http://fonts.googleapis.com/css?family=ZCOOL+XiaoWei"> -->
  1. 在你的 CSS 中，为需要用这个字体的部分的样式选择器添加字体样式：
.className {
  font-family: "Long Cang", serif;
}

方法二引入本地字体文件（加载时间较长）

/* 引入字体文件 */
@font-face {
  font-family: "zfName";
  src: url("xxx.ttf");
}
/* 使用 */
.className {
  font-family: zfName;
}
