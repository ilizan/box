# 初始化&新建项目

node版本大于18.0.0 运行

npx nuxi@latest init <project-name>
例如：
npx nuxi@latest init nuxt-app
1
2
3

如果下载失败 可以修改hosts文件

# 安装依赖

npm i 或者cnpm i

# 启动

npm run dev

浏览器访问 http://localhost:3000

# 是否开启ssr服务端渲染，默认开启

nuxt.config.ts下配置ssr

export default defineNuxtConfig({
    ssr: false
}
