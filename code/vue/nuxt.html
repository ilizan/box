版本

初始化&新建项目

node版本大于18.0.0 运行

npx nuxi@latest init <project-name>
例如：
npx nuxi@latest init nuxt-app

如果下载失败 可以修改hosts文件

安装依赖

npm i 或者cnpm i

启动

npm run dev

浏览器访问 http://localhost:3000

是否开启ssr服务端渲染，默认开启

nuxt.config.ts下配置ssr

export default defineNuxtConfig({
    ssr: false
}

删除根目录下.nuxt文件夹

再运行

npm run dev

则会自动生成.nuxt文件夹

  1. .nuxt 1.server目录里面可以直接写接口 2.app.vue是项目的跟组件 3.nuxt.config.ts是配置文件 4.pages目录是页面文件夹(新建项目默认不带，后面步骤会新建)

app.vue文件中注释，新增

hello world

app.vue中新增路由

<template>
  <!-- Nuxt 基于文件的路由系统 -->

  <!-- 2.首页 index/index.vue -->
  <NuxtLink to="/">首页</NuxtLink>
  <!-- 3.视频页 video/index.vue -->
  <NuxtLink to="/video">视频页</NuxtLink>

  <!-- 1.Nuxt 页面路由，相当于router-view -->
  <NuxtPage />
</template>
