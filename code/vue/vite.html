# Vite

# 简介

Vite 是一个基于 Rollup 的构建工具，它可以将你的项目打包成一个纯 JavaScript 模块，并在开发时与 Rollup 保持一致的速度。

# 快速开始

npm init vite-app my-vue-app
cd my-vue-app
npm run dev
